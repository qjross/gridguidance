Games

High theft risk games - apply 2x tape

High theft risk games/franchises include:

  • Battlefield
  • Call of Duty
  • Grand Theft Auto
  • Halo
  • Madden NFL
  • NBA 2K
  • Super Mario
If you're unsure, it's safer to do the double anyway. Games that require internet activation: toss them if they are not in the original shrink wrap. Assume the donor has already used it.

Desktops

Desktop computers come with:

  • Monitor - which includes power and VGA cords
  • Standard Mouse
  • Standard Keyboard
  • Wireless Antenna (if wi-fi enabled, see below)
Always include these unless customer explicitly declines a part.

If any of them are wi-fi enabled, make sure to run Windows Updates on them regularly because it will save the user a lot of headache later on.

YMMV, but it's recommended to cut the screensaver off not only to make it easier to track when updating, but those with lower specs don't tend to respond well coming off screensaver/sleep mode.

Laptops

ALWAYS MAKE SURE TO HAVE THE CORRECT ADAPTER WITH BOTH BRAND AND VOLTAGE.

Game Consoles

To sell donated consoles, you must provide an HDMI/AV/Component Cable, appropriate power cord, and one controller.

If we do not have these available, just price it and hang it back until we can assemble the bundle.

When someone asks for a PS3, X360, PS4, XBONE controller, assume they want a wireless one.

A rule of thumb for which video cords to sell with consoles:

  • HDMI - XSER, PS5, XBONE, PS4, X360, PS3
  • Keep in mind: some systems have different pinouts for their AV output; Nintendo is notorious for this. HDMI, however, is universal.

PS3, X360, PS4, XBONE, PS5, XSER bundles must have wireless controllers.

TVs

With TVs, test every single port possible.

If a port doesn't work, shave some off the price.

Just remember... nothing under $20 unless it happens to be a CRT TV. Even if a certain port doesn't work, you can use it as a defacto monitor.
