Games

High theft risk games - apply 2x tape

High theft risk games/franchises include:

  • Call of Duty
  • Grand Theft Auto
  • Halo
  • Madden NFL
  • NBA 2K
If you're unsure, it's safer to do the double anyway. Games that require internet activation: toss them if they are not in the original shrink wrap. Assume the donor has already used it.

Desktops

Desktop computers come with:

  • Monitor - which includes power and VGA cords
  • Standard Mouse
  • Standard Keyboard
  • Wireless Antenna (if wi-fi enabled, see below)
Always include these unless customer declines.

If any of them are wi-fi enabled, make sure to run Windows Updates on them because it will save the user a lot of headache later on.

YMMV, but it's recommended to cut the screensaver off not only to make it easier to track, but those with lower specs don't tend to respond well coming off screensaver/sleep mode.
